# Form Testing HW 1
# Требования

Name:
- Латинские буквы
- Большие и маленькие буквы
- Допустимы символы: точка, тире, нижнее подчеркивание
- Max символов 128, min 1 символ
- Обязательное поле
- Нельзя использовать цифры
- При введении большого кол-ва символов поле не должно растягиваться
- При некорректном вводе поле подсвечивается красным (рамка) и высвечивается ошибка

Surname:

- Латинские буквы
- Большие и маленькие буквы
- Допустимы символы: точка, тире, нижнее подчеркивание
- Max символов 128, min 1 символ
- Необязательное поле
- Нельзя использовать цифры
- При введении поле не должно растягиваться
- При некорректном вводе поле подсвечивается красным (рамка) и высвечивается ошибка


Email:

- Max символов 16, min 2 символа до @почта. (тут не уверена)
- Max символов 5, min 2 символа после @почта. (тут не уверена)
- Чувствительность к регистру
- Обязательно поле
- Только латиница
- Можно цифры (?)
- Допустимые спец. символы до @почта. : тире, точка (тут не уверена)
- Допустимые спец. символы после @почта. - только (тут не уверена) буквы


Password:

- Max символов 32, min 5 символов
- Не менее 1 цифры д.б в пароле
- Нижний и верхний регистр
- Можно использовать любой спец.символ кроме пробела
- Латиница
- Должен показывать "точки" когда вводишь пароль
- Должен показывать ошибку при вводе некорректного пароля "Вы неверно ввели пароль"
- Обязательное поле
- Навигация по полям (?)
- Работает ли пароль если вставить через Ctrl+V (должно проходить)
